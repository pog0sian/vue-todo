# Vue To-Do List

Проект To-Do List на стеке Vue 3 + Vite + TypeScript + Tailwind CSS.

## Функционал

- Регистрация и авторизация пользователей
- Создание, редактирование и удаление задач
- Поддержка подзадач
- Фильтрация и отображение прогресса выполнения
- Сохранение данных (Pinia + LocalStorage)

## Установка и запуск

1. Установите зависимости:
   ```bash
   npm install
   ```

2. Запустите режим разработки:
   ```bash
   npm run dev
   ```

3. Сборка для продакшена:
   ```bash
   npm run build
   ```

## Технологии

- Vue 3 (Composition API, `<script setup>`)
- Vite
- TypeScript
- Pinia (State Management)
- Tailwind CSS
- Vue Router
